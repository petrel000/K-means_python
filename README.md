\# 🏥 患者心血管风险分群评估系统



\&gt; 这是一个基于 \*\*K-Means 无监督学习\*\* 的课程学习项目，用于理解聚类算法在医疗数据分析中的应用。



\---



\## 📌 项目简介



作为机器学习初学者，我希望通过这个项目实践：

\- 理解 \*\*K-Means 聚类算法\*\* 的原理和应用

\- 学习 \*\*数据标准化\*\*、\*\*PCA 降维\*\* 等预处理技术

\- 尝试将算法结果与 \*\*医学常识\*\* 结合，进行风险分群

\- 体验完整的前后端开发流程



\---



\## 🔧 技术栈



| 部分 | 技术 |

|------|------|

| 算法核心 | Python + scikit-learn（K-Means、StandardScaler、PCA） |

| 后端 API | Flask |

| 前端展示 | HTML + CSS + JavaScript |

| 数据可视化 | ECharts |



\---



\## 📊 功能演示



\### 已实现

\- \[x] 患者数据 K-Means 自动聚类

\- \[x] 群体特征自动描述（如"高龄、收缩压偏高群体"）

\- \[x] 风险等级划分（高/中/低）

\- \[x] PCA 降维散点图可视化

\- \[x] 新患者风险预测



\### 学习中的局限

\- ⚠️ 使用模拟数据，非真实临床数据

\- ⚠️ 风险判断标准为简化版，仅供算法演示

\- ⚠️ 未经过医学验证，\*\*不能用于实际诊断\*\*
